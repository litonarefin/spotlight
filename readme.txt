== Description ==
Spotlight for WordPress is designed to revolutionize how you navigate and interact with your website. Inspired by the seamless and efficient MacOS Spotlight, this plugin offers you the convenience of performing tasks and finding content with just a few keystrokes.

Whether you're a developer, content creator, or site administrator, Spotlight enhances your productivity by providing instant access to:



- **User Search**: Search Users from Spotlight search
- **Posts and Pages**: Search and open posts and pages without navigating multiple menus (Upcoming)
- **Settings**: Directly access WordPress and plugin settings. (Upcoming)
- **Media**: Quickly find and manage media files. (Upcoming)
- **Custom Post Types**: Search across all your site's custom post types. (Upcoming)
- **And much more!**

Spotlight's elegant and minimalist interface blends seamlessly with your site, with a built-in dark mode for a beautiful, distraction-free experience. It's the perfect tool for anyone looking to speed up their workflow and simplify site management.




== Installation ==

= Manual Installation =

1. **Download the Plugin**:
   Download the latest version of the "Spotlight" plugin from the WordPress.org repository.

2. **Upload the Plugin Files**:
   - Go to your WordPress dashboard.
   - Navigate to **Plugins** > **Add New**.
   - Click **Upload Plugin** and select the downloaded ZIP file.
   - Click **Install Now**.

3. **Activate the Plugin**:
   Once installed, click **Activate**.
   You will now see a "Spotlight" option in your WordPress dashboard to configure settings.

= Using FTP =

1. **Download the Plugin**:
   Download the plugin ZIP file from the WordPress.org repository.

2. **Upload via FTP**:
   - Unzip the file on your computer.
   - Using an FTP client, upload the `spotlight` folder to the `/wp-content/plugins/` directory on your server.

3. **Activate the Plugin**:
   Go to **Plugins** in your WordPress dashboard and click **Activate** under "Spotlight".



== Frequently Asked Questions ==
= How do I open Spotlight in WordPress? =
Once activated, you can launch Spotlight using the customizable keyboard shortcut set in the plugin settings.

= Does Spotlight support dark mode? =
Yes, Spotlight comes with a built-in dark mode that reduces eye strain and provides a beautiful, distraction-free interface.

= Can I search custom post types? =
Absolutely! Spotlight allows you to search across all custom post types registered on your site.
